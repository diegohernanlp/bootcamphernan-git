  1. Que es un sistema de control de versiones?

    Un software para manejar facilmente las versiones de uno o varios archivos

  2. Cual es la diferencia entre git y github?

    Git es el software para el control de versiones y github es la herramienta para administrar un repositorio de git

  3. Cómo creo una rama en git?

    Con el comando git branch creo la nueva rama, debo ademas especificar el nombre de la nueva rama
